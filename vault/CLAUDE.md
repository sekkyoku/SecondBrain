## How To Navigate
1. Start at `_MOC.md` — the master index
2. Follow `[[wiki-links]]` — every link resolves to a real file
3. Read YAML frontmatter — `type`, `tags`, `status`, `related` fields describe each note
4. Check the `_system/` folder for schema definitions, usage instructions, and `LINTING.md`
