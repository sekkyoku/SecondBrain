# ALSAC × Canela Media — Second Brain Vault

## What This Is
A structured knowledge vault for media planning operations. Contains vendor intake data, deal type specs, campaign projections, creative guidelines, targeting capabilities, measurement details, and brief templates for ALSAC × Canela Media.

## How To Navigate
1. Start at `_MOC.md` — the master index
2. Follow `[[wiki-links]]` — every link resolves to a real file
3. Read YAML frontmatter — `type`, `tags`, `status`, `related` fields describe each note
4. Check the `_system/` folder for schema definitions, usage instructions, and `LINTING.md`

## Health Checks
Run `/lint` in Claude Code Desktop to audit this vault for broken links, stale notes, contradictions, and missing concepts. Full spec in `_system/LINTING.md`. Report lands at `_system/lint-report.md`. Treat a `critical` report as a stop-ship for any briefing.
