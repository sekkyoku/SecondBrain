# Digital Buying Guidelines 2025

> Source: `Digital Buying Guidelines 2025.xlsx`
> Context: T-Mobile digital buying guidelines — Canela Media compliance attestation. All guidelines answered Y except IAS blocking (monitoring only on O&O).

---

## Buying Guidelines Summary

### Targeting

- All data sources disclosed as 1P (client), 2P (partner), or 3P (provider). If 3P, provider disclosed. → **Y**

### Distribution

- Inventory sources transparent, curated, authorized; ability to exclude undesired sources → **Y**
- Inclusive site list pre-vetted per brand guidelines → **Y**
- Subdomain variations transparent; concerns addressed with remediation → **Y**
- O&O inventory prioritized; syndication restricted, pre-vetted, approved → **Y**
- Run across approved devices only → **Y**

### Tactics

- OLV instream: no in-banner video, small players, or sticky players; threshold **>400px** → **Y**
- Prioritize OLV instream; in-read/in-feed pre-vetted with specific use case → **Y**
- No invasive/stacking units; users must be able to close → **Y**
- Added value media must meet buying guidelines → **Y**

### Tagging

- Distinct tagging per tactic (video, display, audio, social) → **Y**
- Distinguished tagging for O&O vs. syndication → **Y**
- IAS verification: monitoring on O&O, **blocking on syndication** → **N (monitoring only)**
- IAS-incompatible tactics: alternative reporting accepted → **Y**
- Third-party serving prioritized; site-served only for technical limitations → **Y**
- MMT wrapping for enhanced IAS OLV measurement → **Y**

### Tracking

- Delivery metrics via **DCM** (ad server) + **IAS** (brand safety) → **Y**
- IAS thresholds: site list **100%**, suitability **≥95%**, viewability **≥70%**, fraud **≤1%** → **Y**
- MFA detection by IAS → investigated with makegood → **Y**
- Brand suitability settings adhered to → **Y**
- All impressions must serve within **US Geo**; out-of-geo = non-billable → **Y**
- IAS login set up before campaign launch → **Y**
- IAS tagging limitations disclosed during planning → **Y**

### Brand Safety

- Align to TMO × IAS Brand Safety and Suitability settings → **Y**
- Exclude T-Mobile Arena, MPCS, 5G with negative sentiment → **Y**
- Exclude layoffs/job cuts with negative sentiment → **Y**
- Exclude politics/elections coverage (>50% negative probability) → **Y**
- Within news: prioritize lifestyle/food/entertainment directories → **Y**

### Billing

- Based on 3rd party tracking (DCM or TTD); monthly billable amounts → **Y**

### Separation

- Frequency caps: **1x/hour, 3x/day, 15x/week** (or tighter) → **Y**
- Competitor/category separation: same page or 15-min window → **Y** (O&O when possible)
- No shared category/competitor on sponsorship/tentpole/SOV buys → **Y**
- Makegoods for any buys falling below guidelines → noted

## IAS Brand Suitability Exclusions (13 Rules)

1. Exclude T-Mobile / MPCS / 5G negative sentiment
2. Exclude politics/social issues (>50% negative expected)
3. Exclude layoffs/job cuts negative sentiment
4. Exclude protests/riots/brutality (fear, horror, suffering)
5. Exclude disasters (hurricanes, structural collapses, transit disasters)
6. Exclude corporate culture negatives (lawsuits, discrimination, pay inequality, strikes)
7. Exclude sensitive social issues (immigration, abortion, euthanasia) in negative/partisan context — GARM High Risk
8. Exclude politics with negative emotions (violence, anger, hatred) — note: increases block rate on news
9. Exclude racism/discrimination/offensive language against protected groups
10. Exclude weapon use in news context — GARM Medium Risk
11. Exclude incitement of hatred/hate speech/bullying/bigotry — GARM Low-Floor (all risk levels)
12. Exclude cybersecurity news (hacking, data leaks) — GARM Low-Floor
13. Content in **RED** = T-Mobile cannot surround (absolute exclusion)
