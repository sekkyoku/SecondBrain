## Frequency & Ad Formats

| Field | Detail |
|-------|--------|
| **DNA List (mirroring TV programs)** | Yes — contextual targeting by genre/content category via CAS and direct IO |
| **Avg Length of Ad Pod** | 2 min per pod |
| **Ad Load** | ~15 min/hr (2 min ads per 8 min content block; standard ad load) |
| **Ad Load Changed?** | No — ad load policy unchanged |
| **Frequency Capping** | Yes — by stream and/or time period (standard: **3x/day**) |
| **Skippable Ads** | No — OTT/CTV Pre-Roll and Mid-Roll are **non-skippable**. OLV Outstream is skippable. |
| **Auto Play** | Yes — CTV/OTT auto-play; OLV Outstream auto-play with sound off |

## Tagging & Ad Server

| Field | Detail |
|-------|--------|
| **Internal Ad Servers** | SpringServe (OTT O&O); Google Ad Manager / GAM (Futbol Sites / Online); FreeWheel (TMA, Vidoomy) |
| **VOD via 1x1 Pixels** | Yes — 1x1 tracking pixels accepted for VOD and site-served placements |
| **Ad Server Deal Priority** | DIO → PG → PMP → Open Auction. PG deals hold guaranteed priority in SpringServe waterfall. |
| **Video Ad Lengths** | :06, :15, :30, :60 (must be exact length — not off by 0.01s). Rotation of spots requires separate tags. |
| **Program Level Reports** | Yes — program/content-level reporting available upon request |
| **Tag Restrictions** | VAST tags required (HTTPS/Secure only). 1x1 trackers for display/OLV. **VPAID not supported on OTT. No VPAID.** |
